## üìù HOW TO REPLACE

### Method 1: Direct Replacement (Easiest)
1. Save your school photo with a descriptive name (e.g., `director-photo.jpg`)
2. Upload to `img/` folder
3. Open the HTML file
4. Find the Unsplash/Pexels URL
5. Replace with: `../img/your-photo-name.jpg`
6. Save and refresh browser

### Method 2: Keep Organized
Create subfolders in `img/`:
- `img/heroes/` - for hero banners
- `img/subjects/` - for subject cards
- `img/activities/` - for extracurricular photos
- `img/staff/` - for director and teacher photos

### Example Replacement:
**Before:**
```html
<img src="https://images.unsplash.com/photo-1503676260728-1c00da094a0b" alt="Academics">
```

**After:**
```html
<img src="../img/heroes/academics-hero.jpg" alt="Academics">
```

## ‚ö†Ô∏è IMPORTANT NOTES

1. **Compress all images** before uploading (use tinyjpg.com)
2. **Target file size**: Under 500KB per image
3. **Use descriptive names**: `math-class.jpg` not `IMG_1234.jpg`
4. **Portrait photos**: Ensure faces are visible (not cut off at top)
5. **Clear browser cache** after replacing: `Ctrl + Shift + Delete`
6. **Add version parameter** to force reload: `?v=2` at end of URL
